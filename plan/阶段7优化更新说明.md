# 阶段7优化更新说明

## 更新内容

根据用户反馈，对病历管理功能进行了重新设计和优化：

### 1. 新增PDF上传功能

**后端实现：**
- 新增 `POST /api/v1/documents/upload` 接口
- 支持通过Web界面上传PDF文件
- 上传后自动登记到数据库并索引
- 文件保存到 `D:\webmedic\medical_records\` 目录

**服务层更新：**
- `DocumentService.register_document()` - 登记单个文档
- `IndexService.index_single_document()` - 索引单个文档

### 2. 优化扫描和索引流程

**自动索引：**
- 上传PDF后自动索引
- 扫描本地目录后自动索引
- 无需手动点击"重建索引"

**接口更新：**
- `/api/v1/documents/scan-local` - 扫描后自动索引
- `/api/v1/documents/upload` - 上传后自动索引

### 3. 创建独立的病历管理页面

**新增页面：**
- `frontend/src/views/DocumentManagement.vue`
- 功能完整的病历文档管理界面
- 包含上传、扫描、索引、删除等功能

**页面功能：**
- 📤 上传PDF病历（拖拽或点击上传）
- 📁 扫描本地目录
- 🔄 重建索引
- 📊 文档统计（总数、已索引、待索引）
- 📋 文档列表（文件名、类型、状态、时间）
- 🗑️ 删除文档
- 🔍 单个文档索引

### 4. 添加主导航菜单

**导航结构：**
```
┌─────────────────────────────────────────┐
│  WebMedic Doctor Workbench              │
│  [工作站] [病历管理]          阶段7 v0.1.0│
└─────────────────────────────────────────┘
```

**路由更新：**
- `/` - 医生工作站（原有页面）
- `/documents` - 病历管理（新增页面）

### 5. 优化工作站控制面板

**移除功能：**
- 移除"病历管理"部分
- 移除"扫描本地病历"按钮
- 移除"重建索引"按钮

**保留功能：**
- 会话管理（开始问诊、结束问诊）
- 语音采集（医生说话、患者说话）
- 数据处理（重新抽取、检索相似病历、生成草稿、生成提示）

## 使用流程

### 方式1：上传PDF文件（推荐）

1. 点击顶部导航"病历管理"
2. 点击"上传病历"按钮
3. 拖拽或选择PDF文件
4. 点击"确定上传"
5. 系统自动登记并索引

### 方式2：扫描本地目录

1. 将PDF文件放到 `D:\webmedic\medical_records\` 目录
2. 点击顶部导航"病历管理"
3. 点击"扫描本地目录"
4. 确认目录路径
5. 点击"开始扫描"
6. 系统自动扫描并索引

### 检索相似病历

1. 返回"工作站"页面
2. 开始问诊 → 录入对话 → 重新抽取
3. 点击"检索相似病历"
4. 查看右侧"相似病历"标签页
