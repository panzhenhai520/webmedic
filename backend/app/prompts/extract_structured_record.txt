```json
  "warning_flags": "风险标记（需要特别注意的风险因素、禁忌症等）"
}}
```

## 注意事项
1. 必须输出有效的 JSON 格式
2. 所有字段值必须是字符串类型
3. 如果信息未提及，使用"未提及"
4. 主诉应简洁明了，一般不超过20字
5. 现病史应详细完整，包含时间、症状、程度等要素
6. 初步诊断应基于症状和病史，给出专业的医学判断
7. 建议检查应具体明确，如"血常规、颈椎X光片"等
8. 风险标记应包含需要特别注意的事项，如过敏药物、高危因素等

请直接输出 JSON 对象，不要包含任何其他文字说明。
