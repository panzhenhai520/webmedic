## 新增特性

### 聊天式对话特性

1. **时间排序**：
   - 所有对话按时间戳从早到晚排列
   - 自动滚动到最新消息

2. **视觉区分**：
   - 医生：蓝色气泡（#E3F2FD），左对齐
   - 患者：绿色气泡（#E8F5E9），右对齐
   - 圆角气泡，带边框和阴影

3. **头像显示**：
   - 医生：蓝色头像图标
   - 患者：绿色头像图标

4. **动画效果**：
   - 新消息淡入动画
   - 平滑滚动效果

5. **响应式设计**：
   - 消息气泡最大宽度 75%
   - 自动换行，适应长文本

### 数据排序

```javascript
const sortedDialogues = computed(() => {
  return [...dialogueList.value].sort((a, b) => {
    return new Date(a.timestamp) - new Date(b.timestamp)
  })
})
```
