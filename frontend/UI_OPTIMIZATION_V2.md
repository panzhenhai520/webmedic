### 3. 中间对话区可折叠 ✅

**文件**：`frontend/src/views/DoctorWorkbench.vue`

**功能**：
- 右上角添加折叠按钮（向左箭头）
- 点击后对话区向左滑动隐藏
- 折叠后显示展开按钮（向右箭头）
- 再次点击可展开对话区

**折叠前**：
```
┌────────┬──────────┬──────────┐
│ 左侧   │ 中间对话 │  右侧    │
│ 300px  │  500px   │  (自适应)│
│        │  [←]     │          │
└────────┴──────────┴──────────┘
```

**折叠后**：
```
┌────────┬─┬─────────────────────┐
│ 左侧   │→│      右侧           │
│ 300px  │ │   (更宽，≥700px)    │
│        │ │                     │
└────────┴─┴─────────────────────┘
```

**交互细节**：
- 折叠按钮：圆形，白色背景，悬停时蓝色
- 展开触发器：40px 宽的竖条，悬停时高亮
- 滑动动画：0.3 秒平滑过渡
- 右侧区域自动扩展：min-width 从 500px 增加到 700px

## 技术实现

### 折叠状态管理

```javascript
// 对话区折叠状态
const dialogueCollapsed = ref(false)

// 切换对话区折叠状态
const toggleDialogue = () => {
  dialogueCollapsed.value = !dialogueCollapsed.value
}
```

### 条件渲染

```vue
<!-- 对话区（可折叠） -->
<transition name="slide">
  <el-main v-show="!dialogueCollapsed" class="center-main">
    <el-button class="collapse-button" :icon="ArrowLeft" @click="toggleDialogue" />
    <DialogueArea />
  </el-main>
</transition>

<!-- 展开按钮（折叠时显示） -->
<div v-if="dialogueCollapsed" class="expand-trigger" @click="toggleDialogue">
  <el-button :icon="ArrowRight" />
</div>
```

### 动态样式

```vue
<el-aside
  class="right-aside"
  :style="{
    minWidth: dialogueCollapsed ? '700px' : '500px'
  }"
>
```

### 滑动动画

```css
.slide-enter-active,
.slide-leave-active {
  transition: all 0.3s ease;
}

.slide-enter-from,
.slide-leave-to {
  transform: translateX(-100%);
  opacity: 0;
}
```

## 使用说明

### 折叠对话区
1. 点击对话区右上角的 `←` 按钮
2. 对话区向左滑动隐藏
3. 右侧智能辅助区域自动扩展

### 展开对话区
1. 点击左侧的 `→` 按钮（或点击竖条区域）
2. 对话区从左侧滑入
3. 右侧智能辅助区域恢复原宽度
