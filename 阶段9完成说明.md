## 启动步骤

### 后端启动

1. **确保后端服务正在运行**
```bash
cd D:\webmedic\backend
python run.py
```

2. **验证日志系统**
```bash
# 检查日志文件是否生成
dir logs\

# 查看日志内容
type logs\webmedic.log
```

3. **访问API文档**
- 打开浏览器访问：http://localhost:8001/docs
- 查看新增的会话历史API

### 前端启动

1. **启动前端开发服务器**
```bash
cd D:\webmedic\frontend
npm run dev
```

2. **访问应用**
- 打开浏览器访问：http://localhost:5173
- 点击顶部导航的"会话历史"菜单
