## 🚀 如何使用

### 访问词库维护页面
1. 确保前后端服务运行
2. 打开浏览器访问：http://localhost:5173
3. 点击顶部"系统维护" → "医学词库"
4. 开始使用词库维护功能

### 主要功能操作

#### 查询词汇
- 使用分类筛选：选择"身体部位"、"症状"等
- 使用专科筛选：选择"骨科"、"内科"等
- 使用关键词搜索：输入"肱骨"、"疼痛"等

#### 新增词汇
1. 点击"新增词汇"按钮
2. 填写表单（分类、专科、标准名称、关键词、描述）
3. 系统自动检查相似词
4. 如有相似词，显示警告
5. 点击"保存"完成创建

#### 编辑词汇
1. 点击词汇行的"编辑"按钮
2. 修改需要更新的字段
3. 点击"保存"完成更新

#### 删除词汇
1. 点击词汇行的"删除"按钮
2. 确认删除操作
3. 词汇被软删除（status改为inactive）

## 🎓 学习要点

### 医学词汇分类
1. **身体部位**
   - 骨骼系统：肱骨、桡骨、尺骨、颈椎、腰椎等
   - 关节系统：肩关节、肘关节、髋关节、膝关节等
   - 内脏器官：心脏、肺、肝脏、胆囊、胰腺等

2. **症状**
   - 疼痛、头晕、恶心、呕吐
   - 发热、咳嗽、气短、乏力
   - 失眠、腹泻、便秘、水肿

3. **疾病**
   - 消化系统：胆囊息肉、胆结石、脂肪肝
   - 循环系统：高血压、冠心病
   - 骨骼系统：颈椎病、腰椎间盘突出

4. **方位词**
   - 左、右、上、下
   - 前、后、内、外、中

### ICD-10编码
- 消化系统疾病 (K00-K93)
- 循环系统疾病 (I00-I99)
- 呼吸系统疾病 (J00-J99)
- 肌肉骨骼系统疾病 (M00-M99)
- 内分泌疾病 (E00-E90)

### 手术编码（ICD-9-CM-3）
- 消化系统手术：胆囊切除术、胃镜检查等
- 骨科手术：关节置换术、椎间盘切除术等
- 心血管手术：冠脉搭桥术、心导管检查等
